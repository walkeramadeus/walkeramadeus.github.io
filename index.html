What I've built

Final Year Project
GitHub →

Secrets Posture Manager

A tool for detecting accidentally committed secrets — API keys, tokens, credentials — in Git repositories. Scans both the current working tree and the full commit history using regex pattern matching combined with Shannon entropy analysis to surface high-probability secrets with low false positives.

Built as a practical solution to a real problem: developers push secrets to repos constantly, and most tooling only catches current files. This one goes back through history.
